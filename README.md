# Smart-Greenhouse-System-
STM32F103C8T6 with FreeRTOS collects greenhouse data (temp/humidity), controls lights/motor, and communicates with ESP32 via UART. ESP32 drives touchscreen and uses WiFi/MQTT for remote monitoring/control. Multi-task RTOS ensures real-time operation. Complete IoT solution for smart agriculture.

具体接线信息请前往Smart Greenhouse System.ioc文件查看
本项目材料清单：
stm32f103c8t6最小开发板，esp32开发模块8266无线WIFI+蓝牙双核
发光二极管*1，TB6612FNG电机驱动模块*1,JSHT20温湿度传感器模块*1,光敏传感器模块*4,ST7789-IPS显示屏*1
公对母杜邦线若干，usb数据线，ST-Link下载器
